# Assignments

## Overview

This folder contains the programming assignments completed during the **Object-Oriented Programming** course.

The assignments were designed to reinforce object-oriented programming concepts by solving practical programming problems and implementing real-world programming techniques using **C++**.

## Contents

Each assignment is organized in a separate folder and may include:

- C++ source code
- Assignment report (PDF)
- Supporting files (if applicable)

```text
├── Assignment-01
├── Assignment-02
└── Assignment-03
```
