# Laboratory Tasks

## Overview

This folder contains the laboratory tasks completed during the **Object-Oriented Programming** course.

Each lab focuses on implementing fundamental and advanced object-oriented programming concepts using **C++**. The tasks were designed to strengthen problem-solving skills while applying theoretical concepts through practical programming exercises.

## Topics Covered

- Classes and Objects
- Constructors
- Member Functions
- Encapsulation
- Inheritance
- Polymorphism
- File Handling
- Exception Handling

## Folder Structure

Each laboratory task is organized into its own folder.

```text
Labs
│
├── Lab-01
├── Lab-02
├── Lab-03
├── ...
└── Lab-08
```

Each lab folder may contain:

- C++ source code
- Supporting files (if required)
- Lab report (where available)
