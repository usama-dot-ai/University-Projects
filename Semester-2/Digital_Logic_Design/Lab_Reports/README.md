# Laboratory Reports

## Overview

This folder contains the laboratory reports completed during the **Digital Logic Design** course.

Each laboratory exercise involved designing, simulating, and analyzing digital logic circuits using simulation software while reinforcing theoretical concepts through practical implementation.

## Topics Covered

- Logic Gates
- Boolean Expressions
- Karnaugh Maps
- Combinational Circuits
- Sequential Circuits
- Flip-Flops
- Counters
- Registers
- Multiplexers
- Arithmetic Circuits

## Folder Structure

Each laboratory report is organized into its own folder.

```text
Lab_Reports
│
├── Lab-01
├── Lab-02
├── ...
└── Lab-12
```

Each lab folder may contain:

- Laboratory Report (PDF)
- Simulation Files
- Supporting Resources (if applicable)

## Skills Developed

- Digital Circuit Design
- Logic Circuit Analysis
- Circuit Simulation
- Hardware Problem Solving
