# Digital Logic Design

## Course Overview

This folder contains laboratory reports, assignments, and the semester project completed during the **Digital Logic Design** course in the **2nd Semester** of my **BS Cyber Security** degree at **Air University**.

The course introduced the fundamentals of digital electronics, logic circuit design, and circuit simulation. Practical work involved designing, simulating, and analyzing digital circuits using industry-standard simulation software.

## Topics Covered

- Number Systems
- Boolean Algebra
- Logic Gates
- Combinational Logic Circuits
- Sequential Logic Circuits
- Flip-Flops
- Counters
- Registers
- Multiplexers and Demultiplexers
- Encoders and Decoders
- Arithmetic Logic Circuits
- Digital Circuit Simulation
