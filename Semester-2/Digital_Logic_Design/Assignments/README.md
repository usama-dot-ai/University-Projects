# Assignments

## Overview

This folder contains the assignments completed during the **Digital Logic Design** course.

The assignments focused on designing, analyzing, and simulating digital circuits while applying the theoretical concepts learned during lectures.

## Contents

Each assignment is organized in a separate folder and may include:

- Assignment Report (PDF)
- Simulation Files
- Supporting Documents (if applicable)
