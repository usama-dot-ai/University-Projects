# Computer Networks

## Course Overview

This folder contains laboratory exercises, assignments, and the semester project completed during the **Computer Networks** course in the **3rd Semester** of my **BS Cyber Security** degree at **Air University**.

The course focused on the fundamentals of computer networking, network design, IP addressing, routing, switching, and network simulation using **Cisco Packet Tracer**.

## Topics Covered

- OSI Model
- TCP/IP Model
- IPv4 Addressing
- Subnetting
- Switching
- Routing
- VLANs
- Static Routing
- Dynamic Routing
- DHCP
- DNS
- Network Security Fundamentals
- Cisco Packet Tracer
