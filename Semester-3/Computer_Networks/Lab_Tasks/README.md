# Laboratory Tasks

## Overview

This folder contains the networking laboratory tasks completed during the **Computer Networks** course.

Each laboratory exercise involved designing, configuring, simulating, and troubleshooting computer networks using **Cisco Packet Tracer**.

## Topics Covered

- Network Topology Design
- IPv4 Addressing
- Router Configuration
- Switch Configuration
- VLAN Configuration
- Static Routing
- Dynamic Routing
- DHCP Configuration
- DNS Configuration
- Network Troubleshooting

## Folder Structure

Each laboratory task is organized into its own folder.

```text
Labs
│
├── Lab-01
├── Lab-02
├── ...
└── Lab-XX
```

Each lab folder may contain:

- Cisco Packet Tracer files (.pkt)
- Lab report (PDF)
- Screenshots (if available)

## Skills Developed

- Network Configuration
- Routing and Switching
- Network Troubleshooting
- Simulation using Cisco Packet Tracer
