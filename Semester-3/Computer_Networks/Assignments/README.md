# Assignments

## Overview

This folder contains the assignments completed during the **Computer Networks** course.

The assignments focused on designing, configuring, and analyzing computer networks using Cisco Packet Tracer while applying networking concepts learned throughout the course.

## Contents

Each assignment is organized in a separate folder and may include:

- Assignment Report (PDF)
- Cisco Packet Tracer Files (.pkt)
- Supporting Documents

## Folder Structure

```text
Assignments
│
├── Assignment-01
├── Assignment-02
└── ...
```

## Skills Demonstrated

- Network Design
- Router and Switch Configuration
- IP Address Planning
- Technical Documentation
