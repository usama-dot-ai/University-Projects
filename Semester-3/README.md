# Semester 3

This folder contains the practical coursework completed during **Semester 3** of my **BS Cyber Security** program at **Air University**.

The coursework during this semester focused on networking fundamentals, advanced programming techniques using data structures, and understanding computer architecture through assembly language programming.

## Subjects

- Computer Networks
- Data Structures & Algorithms
- Computer Organization & Assembly Language

- Network Design and Configuration
- IP Addressing and Routing
- Switching Concepts
- Cisco Packet Tracer Simulation
- Data Structures
- Algorithm Design
- Searching and Sorting Algorithms
- Memory Management
- Computer Architecture
- Assembly Language Programming
- Processor Organization
