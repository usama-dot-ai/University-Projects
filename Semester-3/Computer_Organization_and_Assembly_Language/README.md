# Computer Organization & Assembly Language

## Course Overview

This folder contains laboratory tasks, assignments, and the semester project completed during the **Computer Organization & Assembly Language** course in the **3rd Semester** of my **BS Cyber Security** degree at **Air University**.

The course introduced computer architecture fundamentals and low-level programming using assembly language.

## Topics Covered

- Computer Architecture
- Processor Organization
- Registers
- Memory Organization
- Instruction Set Architecture
- Addressing Modes
- Assembly Language Programming
- Interrupts
- Input/Output
